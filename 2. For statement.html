🐍 Trực quan hóa vòng lặp for trong Python

for i in range(n):

📌 Lặp từ 0 đến n-1

for i in range(a, b):

📌 Lặp từ a đến b-1

for i in range(a, b, step):

📌 Lặp từ a đến b-1, bước nhảy step

Giá trị i: -
Số lần lặp: 0
Tổng vòng lặp: 0
Trạng thái: Chờ bắt đầu

📊 Thanh tiến trình giá trị i:

📊 Sơ đồ khối (Flowchart)

💻 Code Python

for i in range(0, 5, 1):
print(i)

📝 Nhấn "Từng bước" hoặc "Tự động" để bắt đầu!
