🐍 Trực quan hóa câu lệnh rẽ nhánh IF trong Python

Dành cho học sinh mới bắt đầu học lập trình

📚 Tóm tắt lý thuyết

📌 Dạng thiếu (if)

if
(điều_kiện):
câu_lệnh

➡️ Nếu điều kiện đúng, thực hiện câu lệnh

📌 Dạng đủ (if-else)

if (điều_kiện):
câu_lệnh_1
else:
câu_lệnh_2

➡️ Nếu đúng → câu_lệnh_1, nếu sai → câu_lệnh_2

Output:

🎯 Mô phỏng trực quan

📊 Sơ đồ khối

1 x = 4
2 if x % 2 == 0 :
3 print ( "x là số chẵn" )
