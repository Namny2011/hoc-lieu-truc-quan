🧮Tính Tổng Từ 1 Đến N

Ví dụ Python: Tính tổng các số từ 1 đến n
sum = 0
for i in range(1, n+1):
    sum = sum + i
Số hiện tại (i)
0
Tổng (sum)
0
Mục tiêu (n)
5
sum = 0
Bước 1: Khởi tạo sum = 0

📋In Ra N Hàng Với Vòng Lặp Lồng Nhau

Ví dụ Python: In ra n hàng, mỗi hàng có các số từ 1 đến n
for i in range(1, n+1):
    for j in range(1, n+1):
        print(j, end=" ")
    print() # Xuống hàng
Hàng hiện tại (i)
0
Cột hiện tại (j)
0
Kích thước (n)
4
Nhấn "Bắt Đầu In" để xem cách in từng hàng!

📄 Kết Quả In Ra:

Chưa bắt đầu...

🔺Bài Tập: In Tam Giác Số

Ví dụ Python: In tam giác số với n hàng
for i in range(1, n+1):
    for j in range(1, i+1):
        print(j, end=" ")
    print() # Xuống hàng
Hàng hiện tại (i)
0
Số hiện tại (j)
0
Điều kiện (j ≤ i)
-
Kích thước (n)
5
Nhấn "Bắt Đầu In" để xem cách tạo tam giác số!

🔺 Kết Quả Tam Giác:

Chưa bắt đầu...

♟️Bài Tập: Bản Cờ Vua

Ví dụ Python: Tạo bản cờ vua n×n với ô đầu tiên là trắng (W)
for i in range(1, n+1):
    for j in range(1, n+1):
        if (i + j) % 2 == 0:
            print("W", end="")
        else:
            print("B", end="")
    print() # Xuống hàng
💡 Giải thích: Dùng (i + j) để ô (1,1) có tổng = 2 (chẵn) → Trắng
Hàng hiện tại (i)
0
Cột hiện tại (j)
0
Tổng (i + j)
0
Màu
-
Kích thước (n)
4
Nhấn "Bắt Đầu Vẽ" để xem cách tạo bản cờ vua!
