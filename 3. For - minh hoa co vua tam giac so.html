🐍 MINH HỌA VÒNG LẶP FOR PYTHON

Học lập trình qua hình ảnh trực quan

📖 CÚ PHÁP VÒNG LẶP FOR

for i in range(n):
    # Khối lệnh thực hiện n lần

🔑 Từ khóa FOR

  • • for = "cho mỗi" - bắt đầu vòng lặp
  • • i = biến đếm (có thể đặt tên khác)
  • • in = "trong" - lặp qua các giá trị

🎯 Hàm RANGE

  • • range(n) → 0, 1, 2, ..., n-1
  • • range(1, n+1) → 1, 2, ..., n
  • • range(start, stop, step)

📊 Ví dụ minh họa range()

range(5)
→ 0, 1, 2, 3, 4
range(1, 6)
→ 1, 2, 3, 4, 5
range(0, 10, 2)
→ 0, 2, 4, 6, 8

6️⃣ VÒNG LẶP FOR LỒNG NHAU CƠ BẢN

🎯 Ý tưởng chính:

Vòng lặp lồng nhau giống như xếp hàng và cột!

  • • i là HÀNG (từ trên xuống ↓)
  • • j là CỘT (từ trái qua phải →)
# Cấu trúc cơ bản:
for i in range(1, n+1): # Hàng
    for j in range(...): # Cột
        # Xử lý ô (i, j)
    print() # Xuống hàng

📊 Cách hoạt động:

  1. 1. Vòng ngoài (i) chọn HÀNG
  2. 2. Vòng trong (j) duyệt CỘT trong hàng đó
  3. 3. Khi j chạy xong → xuống hàng
  4. 4. i tăng lên → lặp lại bước 2
