🧬 Xâu Con Chung Dài Nhất (LCS)

Longest Common Subsequence - Dynamic Programming

01 Phát biểu bài toán

📌 Đầu vào:

  • Xâu X có độ dài m: X = "VANTRUNG"
  • Xâu Y có độ dài n: Y = "HOANGTRONG"

📌 Đầu ra:

  • Xâu con chung dài nhất của X và Y
  • Vị trí các ký tự được chọn trong X và Y

📌 Định nghĩa xâu con:

Xâu con của một xâu là xâu thu được bằng cách xóa đi một số ký tự (có thể không xóa) mà giữ nguyên thứ tự các ký tự còn lại.

📌 Ví dụ minh họa:

X =
V A N T R U N G
Y =
H O A N G T R O N G

✅ Kết quả:

LCS: ANTRNG

Vị trí trong X: 2 3 4 5 7 8

Vị trí trong Y: 3 4 6 7 9 10

02 Ý tưởng Quy hoạch động

📐 Định nghĩa bảng DP:

dp[i][j] = độ dài LCS của X[1..i] và Y[1..j]

🔄 Công thức truy hồi:

dp[i][j] =
0 nếu i = 0 hoặc j = 0
dp[i-1][j-1] + 1 nếu X[i] = Y[j]
max(dp[i-1][j], dp[i][j-1]) nếu X[i] ≠ Y[j]

💡 Giải thích:

1

Trường hợp cơ sở: Nếu một trong hai xâu rỗng, LCS = 0

2

Ký tự giống nhau: Thêm 1 vào kết quả và xét tiếp phần còn lại

3

Ký tự khác nhau: Lấy max của việc bỏ qua ký tự ở X hoặc Y

🔍 Công thức truy vết (từ dp[m][n]):

• Nếu X[i] = Y[j]: lấy ký tự → đi về dp[i-1][j-1]

• Nếu X[i] ≠ Y[j]:

- Nếu dp[i][j] = dp[i][j-1] → đi về dp[i][j-1]

- Ngược lại → đi về dp[i-1][j]

⏱️ Độ phức tạp:

Thời gian: O(m × n)
Bộ nhớ: O(m × n)

03 Minh họa từng bước

🔄 Công thức truy hồi:

dp[i][j] =

• 0, nếu i=0 hoặc j=0

• dp[i-1][j-1]+1, nếu X[i]=Y[j]

• max(dp[i-1][j], dp[i][j-1])

🔍 Công thức truy vết:

Từ dp[m][n]:

• X[i]=Y[j]: lấy ký tự, về dp[i-1][j-1]

• X[i]≠Y[j]:

- dp[i][j]=dp[i][j-1]: về dp[i][j-1]

- ngược lại: về dp[i-1][j]

📍 Trạng thái:

Nhấn "Bước tiếp" để bắt đầu

🎯 Kết quả LCS:

-
Ô đang xét
Đường truy vết
Ký tự được chọn

04 Code C++ hoàn chỉnh

#include <bits/stdc++.h>
#define N 100
using namespace std;

int dp[N+1][N+1];          // Bảng DP
int kqX[N+1], kqY[N+1];    // Đánh dấu vị trí ký tự được chọn
int m, n;                    // Độ dài xâu X và Y
string X, Y, S = "";        // Xâu đầu vào và kết quả

int main() {
    cin >> X >> Y;
    
    // Thêm ký tự đầu để đánh số từ 1
    X = " " + X;
    Y = " " + Y;
    m = X.size() - 1;
    n = Y.size() - 1;
    
    // ═══════════════════════════════════════
    // PHẦN 1: ĐIỀN BẢNG DP
    // ═══════════════════════════════════════
    for (int i = 1; i <= m; i++) {
        for (int j = 1; j <= n; j++) {
            if (X[i] == Y[j]) 
                // Ký tự giống: thêm 1 vào kết quả
                dp[i][j] = dp[i-1][j-1] + 1;
            else 
                // Ký tự khác: lấy max
                dp[i][j] = max(dp[i-1][j], dp[i][j-1]);
        }
    }

    // ═══════════════════════════════════════
    // PHẦN 2: TRUY VẾT TÌM XÂU CON
    // ═══════════════════════════════════════
    int i = m, j = n;
    while (i > 0 && j > 0) {
        if (X[i] == Y[j]) {
            // Ký tự giống: thêm vào kết quả
            S = X[i] + S;
            kqX[i] = kqY[j] = 1;  // Đánh dấu vị trí
            i--; j--;              // Đi chéo
        }
        else if (dp[i][j] == dp[i][j-1]) 
            j--;                   // Đi sang trái
        else 
            i--;                   // Đi lên trên
    }

    // ═══════════════════════════════════════
    // PHẦN 3: IN KẾT QUẢ
    // ═══════════════════════════════════════
    cout << "LCS: " << S << "\n";
    
    cout << "Vị trí trong X: ";
    for (int i = 1; i <= m; i++)
        if (kqX[i] == 1) cout << i << " ";
    cout << "\n";
    
    cout << "Vị trí trong Y: ";
    for (int i = 1; i <= n; i++)
        if (kqY[i] == 1) cout << i << " ";
    
    return 0;
}

📊 Phần 1: Điền bảng

Duyệt qua từng cặp (i,j), áp dụng công thức truy hồi để tính dp[i][j].

🔍 Phần 2: Truy vết

Đi ngược từ dp[m][n], tìm các ký tự thuộc LCS và đánh dấu vị trí.

📝 Phần 3: Kết quả

In xâu LCS và các vị trí ký tự được chọn trong X và Y.

// Ví dụ với X = "VANTRUNG", Y = "HOANGTRONG"

Input: VANTRUNG HOANGTRONG

Output:

LCS: ANTRNG

Vị trí trong X: 2 3 4 5 7 8

Vị trí trong Y: 3 4 6 7 9 10
